△ Architecture

Built for the terminal, designed for humans

TUI Shell
Layout · Editor · Modals
Agent Core
Orchestration · Context
LLM Provider
Anthropic · OpenAI · Google
Tool Executor
Approval · Sandbox
Conversation
History · Multi-turn

Glue's architecture is built on Dart's async runtime, giving you non-blocking I/O across every layer — from TUI rendering to model streaming to file operations.

The Agent Core orchestrates conversations, manages tool execution with approval gates, and maintains context across multi-turn interactions. LLM providers are pluggable — swap between any provider with a single flag.
