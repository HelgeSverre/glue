The coding agent that holds it all together.
Open source CLI coding agent built in Dart.
Multi-model. Multi-worktree. Impossibly fast.
$ dart pub global activate glue
SYSTEM CAPABILITIES
  • --fast Built in Dart for native speed. No Node.js, no Python runtime overhead. Starts instantly, responds immediately.
  • --worktree Run parallel agents across git worktrees. Ship features simultaneously without context switching.
  • --model <any> Anthropic, OpenAI, Google, local models. Swap providers with a flag. Use the best model for each task.
  • --approve Every file write, every command requires your explicit approval. You stay in control at all times.
  • --mcp First-class Model Context Protocol support. Extend Glue with any MCP server for custom tool integrations.
  • --zero-config Drop into any project and start coding. Glue figures out the rest — languages, frameworks, project structure.
