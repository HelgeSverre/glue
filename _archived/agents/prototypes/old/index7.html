Speed

Dart Runtime Delivers Native Performance

No Node.js, no Python overhead. Glue starts instantly and responds immediately, leveraging Dart's compiled speed and async runtime for non-blocking I/O across every layer.

Flexibility

Any Model, Any Provider, One Flag

Swap between Anthropic, OpenAI, Google, or any OpenAI-compatible endpoint. No restart required. Use the best model for each task without configuration changes.

Control

Every Action Requires Your Approval

File writes, command execution — nothing happens without explicit consent. Glue's approval gates keep you in command while the agent handles the heavy lifting.

Feature Report

1

Blazing Fast

Built in Dart for native speed. No runtime overhead. Starts instantly, responds immediately. Compiled performance for real-time interaction.

2

Multi-Worktree

Run parallel agents across git worktrees. Ship features simultaneously without context switching. True concurrent development.

3

Any Model

Anthropic, OpenAI, Google, local models. Swap providers with a flag. Use the best model for the task at hand.

4

Tool Approval

Every file write, every command requires explicit approval. You remain in control at all times. Safety by design.

5

MCP Native

First-class Model Context Protocol support. Extend Glue with any MCP server for custom tool integrations and workflows.

6

Zero Config

Drop into any project and start coding. Glue figures out the rest — languages, frameworks, project structure. No setup required.
