Features

Everything you need. Nothing you don't.

Glue is opinionated about simplicity. No bloated IDE plugins, no cloud dependencies — just a fast CLI that gets out of your way.

🔥

Blazing Fast

Built in Dart for native speed. No Node.js, no Python runtime overhead. Starts instantly, responds immediately.

🔀

Multi-Worktree

Run parallel agents across git worktrees. Ship features simultaneously without context switching.

🤖

Any Model

Anthropic, OpenAI, Google, local models. Swap providers with a flag. Use the best model for each task.

🛡️

Tool Approval

Every file write, every command requires your explicit approval. You stay in control at all times.

🔌

MCP Native

First-class Model Context Protocol support. Extend Glue with any MCP server for custom tool integrations.

📦

Zero Config

Drop into any project and start coding. Glue figures out the rest — languages, frameworks, project structure.

Under the Hood

Built for the terminal, designed for humans

TUI Shell ◄──► Agent Core ◄──► LLM Provider │ │ ▼ ▼ Layout Tool Executor Editor Conversation Modals History

Glue's architecture is built on Dart's async runtime, giving you non-blocking I/O across every layer — from TUI rendering to model streaming to file operations.

The Agent Core orchestrates conversations, manages tool execution with approval gates, and maintains context across multi-turn interactions. The TUI Shell provides a responsive terminal interface with rich layouts, inline editing, and modal dialogs.

LLM providers are pluggable — swap between Anthropic, OpenAI, Google, or any OpenAI-compatible endpoint with a single flag. No restart required.
