◆ Architecture

Built for the terminal, designed for humans

TUI Shell ◄──► Agent Core ◄──► LLM Provider │ │ ▼ ▼ Layout Tool Executor Editor Conversation Modals History

Glue's architecture is built on Dart's async runtime, giving you non-blocking I/O across every layer.

The Agent Core orchestrates conversations, manages tool execution with approval gates, and maintains context. LLM providers are pluggable — swap with a single flag.
